Acerca de

Weblate es una herramienta de traducción basada en la web con una intensa integración a Git. Posee una interfaz limpia y simple, propagación de traducciones en subproyectos, chequeos de consistencia y asociación automática a archivos de código fuente.

Lista de características incluidas:

  • Traducción sencilla en la web
  • Propagación de traducciones entre subproyectos (para diferentes ramas)
  • Fuerte integración con git - todo cambio es representado como un commit en Git
  • Uso de interfaz de administrador de Django
  • Subida y fusionado automático de archivos de traducción
  • Enlaces a los archivos fuente para contextualizar
  • Permite el uso de servicios de traducción mecanizados
  • Comprueba la consistencia del mensaje
  • Control de acceso personalizable
  • Admite un amplio rango de formatos de traducción (Gettext, Qt, Java, Windows, Symbian y más)

Si utiliza Git como sistema de control de versiones Weblate la herramienta de «CAT» (traducción asistida por computadora según sus siglas en inglés).

Descargar

Puede descargarse la última versión en Github.

Para usuarios de openSUSE/SLES, hay paquetes binarios disponibles en openSUSE Buildservice.

En caso que desee ejecutar Weblate en su servidor virtual o nube puede descargar la aplicación basada en openSUSE 12.1 desde la SUSE Studio Gallery.

Código

Weblate es desarrollado en Github, puede hacer derivados allí mismo o simplemente: 

git clone git://github.com/nijel/weblate.git
