关于

Weblate 是一个高度 Git 集成的在线翻译工具。它提供简洁清爽的用户界面，子项目间翻译同步，一致性检查以及自动链接到源文件等功能。

详细功能有:

  • 方便的在线翻译
  • 子项目间翻译同步 (在不同分支上)
  • 高度 git 集成 - 每次修改都通过 Git 提交
  • 使用 Django 管理界面
  • 上传及自动合并 po 文件
  • 链接到源文件获得上下文
  • 机器翻译服务参考
  • 消息一致性检查
  • 可调访问控制
  • 大量格式支持 (Gettext, Qt, Java, Windows, Symbian 以及更多)
