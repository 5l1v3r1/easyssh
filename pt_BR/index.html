Sobre

Weblate é uma ferramenta de tradução baseada na web, com uma intensa integração com o Git. Ele possui uma interface limpa e simples, propagação de traduções em subprojetos, verificação de consistência e associação automática aos arquivos fontes.

Lista de funcionalidades inclusas:

  • Tradução fácil pela web
  • Propagação de traduções através dos subprojetos (nas diferentes ramificações)
  • Integração forte com o git - cada mudança é representada como uma publicação no Git
  • Uso da interface de administração do Django
  • Enviar e mescla automática dos arquivos de tradução
  • Links para os arquivos fontes para contextualizar
  • Permite usar serviços de traduções automáticas
  • Verificação de consistência de mensagem
  • Controle de acesso personalizável
  • Ampla variedade de formatos de arquivos de tradução (Gettext, Qt, Java, Windows, Symbian e outros)

Se você está usando o GIT como sistema de controle de versão, o Weblate é a ferramenta CAT (tradução assistida por computador) certa para você.

Descarregar

Você pode baixar a última versão do Github.

Para usuários de openSUSE/SLES, existem pacotes binários disponíveis em openSUSE Buildservice.

No caso de que queira executar o Weblate na sua nuvem ou servidor virtual, pode baixar a amplicação baseada em openSUSE 12.2 em SUSE Studio Gallery.

O Weblate também está disponível como BitNami stack, o qual permite a você uma instalação nativa no Windows, Linux ou Mac OSX e provê imagens virtuais para rodar o Weblate.

Código

O Weblate está sendo desenvolvido em Github, e você pode copiá-lo ali mesmo ou simplesmente: 

git clone git://github.com/nijel/weblate.git
