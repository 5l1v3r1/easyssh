O Weblate

Weblate to narzędzie do tłumaczenia z interfejsem WWW, zintegrowane z systemem kontroli wersji Git. Posiada przejrzysty interfejs użytkownika, możliwość scalania tłumaczeń pomiędzy podprojektami, sprawdzanie spójności i automatyczne odnośniki do plików źródłowych.

Krótka charakterystyka:

  • Łatwe tłumaczenie na stronach WWW
  • Wymiana tłumaczeń pomiędzy podprojektami (dla różnych gałęzi)
  • Ścisła integracja z gitem - każda zmiana reprezentowana jest jako wpis w repozytorium Git
  • Wykorzystanie interfejsu admina Django
  • Przesyłanie i automatyczne scalanie plików .po
  • Odsyłacze do plików źródłowych w celu ustalenia kontekstu
  • Możliwość wykorzystania automatycznych serwisów tłumaczących
  • Testy na spójność komunikatu
  • Możliwość dostosowania kontroli dostępu
  • Szeroki zakres wspieranych formatów tłumaczeń (Gettext, Qt, Java, Windows, Symbian, i inne)

Pobierz

Najnowszą wersję można pobrać z: Github.

Dla użytkowników openSUSE/SLES, przygotowane zostały pakiety binarne, dostępne w openSUSE Buildservice.

Jeśli chcesz uruchomić Weblate w chmurze lub na serwerze wirtualnym, możesz pobrać obraz oparty na openSUSE 12.1 z SUSE Studio Gallery.

Kod źródłowy

Weblate jest rozwijany na Github, możesz tam stworzyć własną gałąź lub po prostu: 

git clone git://github.com/nijel/weblate.git

Dokumentacja

Dokumentacja jest dostępna w postaci kodu źródłowego w katalogu docs lub online na stronie Czytaj Dokumentację.
