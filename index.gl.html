Sobre

Weblate é unha ferramenta de tradución en web estreitamente integrada con Git. Ten unha interface simple e limpa, propagación de traducións a través de proxectos, probas de consistencia e ligazón automática cos ficheiros de orixe.

Lista de características incluídas:

  • Tradución sinxela por web
  • Propagación de traducións a través dos subproxectos (nas diferentes ramificacións)
  • Estreita integración con git - cada cambio represéntase mediante unha remisión de Git
  • Uso da interface de administración Django
  • Envío e fusión automática de ficheiros PO
  • Liga cos ficheiros de orixe para contexto
  • Permite usar servizos de tradución automática
  • Probas de consistencia das mensaxes
  • Control de acceso afinábel
  • Ampla variedade de formatos de ficheiros de tradución (Gettext, Qt, Java, Windows, Symbian e outros)

Descargar

Pode descargar a última versión en Github.

Para usuarios de openSUSE/SLES, hai paquetes binarios dispoñíbeis en openSUSE Buildservice.

No caso de que queira executar Weblate na súa nube ou servidor virtual, pode descargar a ampliación baseada en openSUSE 12.1 en SUSE Studio Gallery.

Código

Weblate desenvólvese en Github, pode facer derivados alí mesmo ou só: 

git clone git://github.com/nijel/weblate.git
