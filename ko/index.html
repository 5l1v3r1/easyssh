소개

Weblate is a web based translation tool with tight Git integration. It features simple and clean user interface, propagation of translations across subprojects, quality checks and automatic linking to source files.

포함하고 있는 기능:

  • 쉬운 웹 기반 번역
  • 하위 프로젝트간 번역 전달(다른 Git 브랜치간)
  • git와 밀접한 통합 - 모든 변경사항은 Git 커밋으로 표시됨
  • Django의 관리 인터페이스 사용
  • 번역 파일의 업로드 및 자동 통합
  • 문맥 파악을 위한 원본 파일 연결
  • 기계 번역 서비스 사용이 가능함
  • Message quality checks
  • 접근 제어 조정가능
  • 폭 넓은 번역 포맷 지원 (Gettext, Qt, Java, Windows, Symbian 등등)

만약 Git를 버젼 제어 체계로 사용한다면, Weblate는 적절한 CAT (컴퓨터로 보조된 번역) 도구입니다.

다운로드

최종 배포판은 Github에서 다운로드 할 수 있습니다.

openSUSE/SLES 사용자들은, openSUSE Buildservice에서 바이너리 패키지를 구할수 있습니다.

Weblate를 클라우드나 가상서버에서 실행하고자 하는 경우, openSUSE 12.2 기반의 어플라이언스를SUSE Studio Gallery에서 받을수 있습니다.

Weblate는 BitNami stack으로서도 구할수 있으며, Windows, Linux or Mac OS X에서 직접 설치하거나, Weblate를 실행 가능한 가상 이미지 파일 형식으로 구할수 있습니다.

Weblate는 Github에서 개발되고 있습니다, 당신은 그 곳에서 포크하거나 아니면: 

git clone git://github.com/nijel/weblate.git

문서

문서는 소스상의 docs 디렉터리나 온라인상의 Read the Docs에서 찾을수 있습니다.

법적 정보

Weblate는 Michal Čihař에 의해 작성되었고 자유 소프트웨어 입니다: 자유 소프트웨어 재단에 의해서 공표된 GNU 일반 공중 허가서 버전 3 또는 (귀하의 선택으로) 이후 버전의 조항에 따라 재배포 하거나 수정할 수 있습니다.

여러 분야의 프로젝트가 Weblate를 사용합니다. 그 중의 일부가 이 정보를 공개하고 있으며 여기에서 볼 수 있습니다:

l10n.cihar.com
phpMyAdmin, Gammu와 Weblate를 번역.
LinuxCNC의 웹 번역

데비안과 연관된 번역 서버는 현재 The Debian Administrator's Handbook를 번역하고 있습니다.

Osmand (OSM Automated Navigation Directions) 번역.
