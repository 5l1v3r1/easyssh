Περί

Το Weblate είναι ένα διαδικτυακό εργαλείο μετάφρασης με ενσωμάτωση Git. Διαθέτει καθαρό και απλό περιβάλλον χρήστη, διαμοιρασμός των μεταφράσεων μεταξύ υποέργων, ακέραιους ελέγχους και αυτόματη σύνδεση με τα πηγαία αρχεία.

Η λίστα χαρακτηριστικών περιλαμβάνει:

  • Εύκολη διαδικτυακή μετάφραση
  • Διαμοιρασμός των μεταφράσεων μεταξύ υποέργων (για διαφορετικούς κλάδους)
  • Ενσωμάτωση git - κάθε αλλαγή αντιπροσωπεύεται σε καταχώρηση Git
  • Χρήση περιβάλλοντος διαχείρισης Django
  • Αποστολή και αυτόματη ένωση των αρχείων μετάφρασης
  • Συνδέσεις προς τα πηγαία αρχεία για το περιεχόμενο
  • Επιτρέπει τη χρήση υπηρεσιών μηχανικής μετάφρασης
  • Έλεγχοι ακεραιότητας μηνύματος
  • Ευέλικτος έλεγχος πρόσβασης
  • Ευρύ φάσμα υποστηριζόμενων μορφών μετάφρασης (Gettext, Qt, Java, Windows, Symbian και άλλα)

Αν χρησιμοποιείτε το Git ως σύστημα ελέγχου έκδοσης, το Weblate είναι το σωστό εργαλείο υποβοηθούμενης μετάφρασης από υπολογιστή για σας.

Λήψη

Μπορείτε να λάβετε την τελευταία έκδοση στο Github.

Για χρήστες openSUSE/SLES, υπάρχουν διαθέσιμα εκτελέσιμα πακέτα στο openSUSE Buildservice.

Σε περίπτωση που θέλετε να εκτελέσετε το Weblate στο νέφος σας ή στο διακομιστή σας, μπορείτε να πάρετε συσκευή που βασίζεται σε openSUSE 12.1 στη Συλλογή SUSE Studio.

Κώδικας

Το Weblate αναπτύσσεται στο Github, μπορείτε να δημιουργήσετε διακλάδωση ή απλά: 

κλώνος git git://github.com/nijel/weblate.git

Τεκμηρίωση

Η τεκμηρίωση είναι διαθέσιμη στον κατάλογο docs στα πηγαία αρχεία ή διαδικτυακά στο Διάβασε τα έγγραφα.

Δοκιμάστε το

Μπορείτε να δοκιμάσετε πως λειτουργεί το Weblate στον δοκιμών διακομιστή. Μπορείτε να εγγραφείτε εκεί και να εργαστείτε κάτω από το όνομα χρήστη σας ή απλά να χρησιμοποιήσετε λογαριασμό demo με συνθηματικό το demo.

Εφαρμογές

Εάν θέλετε να δοκιμάσετε πιο βαθιά, μπορείτε να είτε να λάβετε έτοιμο στιγμιότυπο από τη Συλλογή SUSE Studio ή χρησιμοποιήστε τη δοκιμή όπου σας επιτρέπει να δοκιμάσετε τον εικονικό διακομιστή για μία ώρα.

Weblate is also available as BitNami stack, what allows you native installation on Windows, Linux or Mac OS X and provides virtual images for running Weblate.
