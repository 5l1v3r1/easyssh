A propos

Weblate est un outil de traduction sur le web fortement lié avec Git. Il comporte une interface utilisateur simple, permet la propagation de traduction à travers les sous-projets, permet de vérifier la cohérence des traductions et permet de se lier automatiquement aux fichiers sources.

Les fonctionnalités comprennent :

  • Traduction facile sur le web
  • Propagation des traductions à travers les sous-projets (pour différentes branches)
  • Intégration forte à git - chaque changement est représenté par un commit git
  • Utilise l'interface d'administration de Django
  • Téléversement et fusion automatique des fichiers po
  • Liens avec les fichiers sources pour le contexte
  • Permet d'utiliser des services de traductions automatisés
  • Vérification de conformité des messages
  • Contrôle d'accès configurable
  • Large éventail de formats de traduction supportés (Gettext, Qt, Java, Windows, Symbian et plus encore)

Télécharger

Vous pouvez télécharger la dernière version sur Github.

Pour les utilisateurs d'openSUSE/SLES, il y a un paquet binaire disponible sur le Buildservice d'openSUSE.

Si vous souhaitez faire tourner Weblate dans votre nuage ou sur votre serveur virtuel, vous pouvez télécharger l'image disque basée sur openSUSE 12.1 sur le SUSE Studio Gallery.

Weblate est développé sur Github, vous pouvez le forker ou simplement :
